Environmental Statistics – Instructor Analysis Key

Instructor Note

Each dataset begins with data visualization. Emphasize to students that plots come before statistical tests.

Dataset 1: Urban Heat & Vegetation

Visualize Data

d <- read.csv('dataset1_urban_heat.csv')
plot(d$Vegetation_Percent, d$Surface_Temperature_C,
     xlab = 'Vegetation Cover (%)',
     ylab = 'Surface Temperature (C)',
     pch = 19)
abline(lm(Surface_Temperature_C ~ Vegetation_Percent, data = d), col = 'red')

Analysis: Linear Regression

lm(Surface_Temperature_C ~ Vegetation_Percent, data = d)
## 
## Call:
## lm(formula = Surface_Temperature_C ~ Vegetation_Percent, data = d)
## 
## Coefficients:
##        (Intercept)  Vegetation_Percent  
##           38.32675            -0.07606

Interpretation: Negative slope indicates cooling effect of vegetation.

Dataset 2: Water Quality

Visualize Data

d <- read.csv('dataset2_water_quality.csv')
boxplot(Nitrate_mgL ~ Location, data = d,
        ylab = 'Nitrate (mg/L)', xlab = 'Location')

Analysis: Two-sample t-test

t.test(Nitrate_mgL ~ Location, data = d)
## 
##  Welch Two Sample t-test
## 
## data:  Nitrate_mgL by Location

##                   2.9856                   2.0988

Dataset 4: Hydrology

Visualize Data

d <- read.csv('dataset4_hydrology.csv')
plot(d$Rainfall_mm, d$Streamflow_m3s,
     xlab = 'Rainfall (mm)',

Analysis: Correlation

cor.test(d$Rainfall_mm, d$Streamflow_m3s)

Dataset 5: Biodiversity

Visualize Data

d <- read.csv('dataset5_biodiversity.csv')

## 
##  Kruskal-Wallis rank sum test
##

Dataset 6: Air Quality Policy

Visualize Data

d <- read.csv('dataset6_air_quality.csv')
boxplot(d$PM25_Before, d$PM25_After,
        names = c('Before', 'After'),
